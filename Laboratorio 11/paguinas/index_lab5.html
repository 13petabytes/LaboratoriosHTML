Diferencias y semejanzas entre Java y JavaScript:

Como semejanzas tienen sus sintanxis, uso de llaves if, for, etc, y su uso en desarrollo web.

Como diferencias tienen su paradigma, Java esta muy orientado al uso de objetos, mientras que JavaScript es más flexible, Java de ejecuta en JVM y JavaScript en navegadores Web y por ultimo, si bien tiene semejanzas en la sintaxis, la misma presenta diferencias.

Métodos del objeto Date:

getFullYear() → Devuelve el año en formato de 4 dígitos.

getMonth() → Devuelve el mes (0 para enero, 11 para diciembre).

getDate() → Devuelve el día del mes (1-31).

getDay() → Devuelve el día de la semana (0 para domingo, 6 para sábado).

getHours() → Devuelve la hora actual (0-23).

Métodos de los arreglos en JavaScript:

push(elemento) → Agrega un elemento al final del arreglo.

pop() → Elimina el último elemento del arreglo.

shift() → Elimina el primer elemento del arreglo.

unshift(elemento) → Agrega un elemento al inicio del arreglo.

map(función) → Crea un nuevo arreglo aplicando una función a cada elemento.

Declaración de una variable con alcance local dentro de una función:

Se emplea "let" o "const" como parametro para la variable

Implicaciones de usar variables globales dentro de funciones:

El uso execivod e variabes globales puede causar problemas a la hroa de diceñar el programa, ya que multiples funciones estarian modificando una sola variable, difucultando la depuración, aunado a los problemas de rendimiento que genera.
